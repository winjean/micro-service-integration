 * Zuul  
Zuul的主要功能是路由转发和过滤器  
zuul默认和Ribbon结合实现了负载均衡的功能  
zuul有以下功能：  
Authentication  
rate limit --- spring-cloud-zuul-ratelimit  
Insights  
Stress Testing  
Canary Testing  
Dynamic Routing  
Service Migration  
Load Shedding  
Security  
Static Response handling  
Active/Active traffic management
